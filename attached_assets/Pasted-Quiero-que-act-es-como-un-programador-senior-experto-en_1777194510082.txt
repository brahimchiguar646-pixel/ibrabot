Cómo conectar APIs y mantener control
Principios

Usa tokens que tú generes y guardes en el secrets manager de Replit o GitHub Actions Secrets.

Nunca hardcodees credenciales.

Todas las acciones que impliquen cambios en cuentas externas deben requerir un comando de confirmación tuya (ejemplo: !confirm canva create design <id>).

Canva

Si Canva tiene API pública: crea una app en Canva, genera token, guarda en CANVA_API_KEY en secrets. El módulo integrations/canva.js usará ese token para crear diseños.

WhatsApp

Usa WhatsApp Business API o proveedores oficiales (Twilio, Meta Business). Registra un número Business, guarda credenciales en secrets, configura webhook para recibir mensajes y endpoints para enviar mensajes.

Despliegue y control

Añade un panel web mínimo (/admin) protegido por auth para aprobar acciones del agente y ver logs.

Mantén un audit log en logs/ con cada acción del agente y la confirmación del usuario.
